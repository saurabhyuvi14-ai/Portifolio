# Saurabh Kumar | Premium Developer Portfolio

A futuristic, high-end personal portfolio website built with HTML, CSS, and Vanilla JavaScript.

## Features
- **Animations**: GSAP, ScrollTrigger, and Lenis for ultra-smooth scrolling and cinematic reveals.
- **3D Elements**: Three.js floating background shapes with hardware-accelerated movement.
- **Design**: Glassmorphism UI, interactive magnetic buttons, and responsive layouts.
- **Projects**: Showcasting robotics (Battle Robo Car) and web applications.
- **Interactivity**: Particles.js background and typing effects.

## Tech Stack
- HTML5 / CSS3 / JavaScript
- Three.js
- GSAP (GreenSock Animation Platform)
- Lenis Smooth Scroll
- Particles.js
- Font Awesome

## Getting Started
1. Clone the repository
2. Open `index.html` in your browser or serve locally using `python3 -m http.server 8000`.
